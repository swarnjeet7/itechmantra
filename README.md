## Switching between color themes

```css

--secondaryTextColor: (--secondaryTextColor-light);
--mainLinkColor: var(--mainLinkColor-light);
--mainBorderColor: var(--mainBorderColor-light);
--mainBgColor: var(--mainBgColor-light);
```
